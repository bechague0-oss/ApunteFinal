Arduino Uno

Plataforma de Electrónica

Arduino es una plataforma de electrónica de código abierto basada en hardware y software fácil de usar. Es ideal para principiantes y expertos en proyectos de robótica, domótica y automatización. Permite crear proyectos interactivos que combinan hardware y software.

Componentes Principales

  • Microcontrolador programable
  • Entradas y salidas digitales/analógicas
  • Puertos USB para programación
  • IDE basado en Processing
  • Amplia comunidad y documentación
  • Variedad de shields y módulos

Proyectos Comunes

Con Arduino puedes crear:

  • Sistemas de riego automático
  • Estaciones meteorológicas
  • Robots controlados por Bluetooth
  • Sistemas de seguridad
  • Interfaces hombre-máquina
  • Medidores y sensores
  • Controles por telecomando

Lenguaje de Programación

Arduino utiliza una variante de C++ simplificada que es fácil de aprender. La estructura básica incluye dos funciones principales: setup() para la inicialización y loop() para el código que se ejecuta continuamente.
